# 云枢工具系统评估发现

## 可用性问题
- 返回格式不一致（字符串/dict/混合）
- web_search 结果仅返回 3 条，限制过严
- ext_list 中存在乱码注释
- search_files 缺少路径安全校验
- 错误信息中英文混杂

## 功能强度不足
- 无异步/流式工具执行
- 无定时调度工具
- 无数据库操作工具
- 无图像处理工具
- 无文件压缩/解压工具
- 无文件比较/diff 工具
- 无 JSON/YAML 处理工具
- 无数据导出工具
- 无截图工具
- 无工具调用限流
- 无工具依赖管理
- web_search 返回深度不足
- 缺少多引擎搜索去重合并

## 稳定性问题
- 仅注册表框架有测试，具体工具集成测试覆盖率为 0
- 网络工具无 Mock 测试
- 技能开关状态同步问题（3个数据文件互相覆盖）
- 扩展管理器多重实例化（无复用）
- Persona/Preferences 条件注册工具对 LLM 不透明
- web_search 引擎初始化对外部配置依赖强
