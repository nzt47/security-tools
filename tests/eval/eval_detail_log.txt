【六、根因诊断与升级建议】
----------------------------------------------------------------------------------------------------
  1. 核心根因：self_reflection 与 memory_summary 的 skill.md front matter description 字段为空
     → TF-IDF 第一层只读 front matter（name + description + tags + category）
     → 这两个技能 name 是英文 ID，description 为空，tags 为空，category=custom
     → 中文语义查询无任何字面命中点

  2. 次要根因：TF-IDF 排序缺陷
     → case_026 '我想用语音跟你说话'：voice_interaction 与 proactive_suggestion 同分 0.3333
       因 proactive_suggestion 字典序靠前被排到前面（实际算法用 stable sort）
     → case_012 '请调整回应语气加点感情'：context_aware 排第一（0.4545）
       因 '回应' 同时出现在 context_aware 与 emotion_expression 的 description 中

  3. 升级向量检索的预期收益
     → 7 个 0 分用例全部修复：Precision@3 从 0.3926 → 0.5481（+0.1555）
     → 加上排序优化（case_012、case_026 等恢复 Top-1 命中）：Precision@3 → ~0.6+
     → MRR 从 0.8000 → ~0.95（7 个用例 MRR 从 0 提升到 1.0）

  4. 短期缓解（不改算法）：补全 self_reflection / memory_summary 的 description 字段
     → 但【不易】约束禁止改技能定义，所以必须走升级向量检索路径
