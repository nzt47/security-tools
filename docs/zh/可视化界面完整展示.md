# 云枢数字生命体 - 可视化界面完整展示

## 项目概览
**云枢**是一个具有数字生命的智能助手系统，采用AIRI+东方禅意的设计风格。所有界面均围绕"有呼吸感的数字生命"这一核心理念设计。

---

#### 情绪状态视觉表现
| 状态 | 眼睛 | 嘴巴 | 光晕颜色 | 动画 |
|------|------|------|----------|------|
| idle | 正常 | ◡ | 云枢青 | 缓慢呼吸 |

- **情绪切换**：根据应用状态改变表情
- **悬停反馈**：悬停时轻微放大

### 14. Toast通知组件

#### 组件名称：轻量级通知提示
#### 视觉结构
```
┌─────────────────────────────────────┐
│ ✓ │  操作成功！                     │ × │
│   │  配置文件已保存                  │   │
│   ├─────────────────────────────────│   │
│   │ ▓▓▓▓▓▓▓▓▓▓░░░░░░░░░░ (进度条) │   │
└───┴─────────────────────────────────┘
```

#### 类型变体
- **success**: 绿色左边框 + ✓ 图标
- **warning**: 橙色左边框 + ⚠ 图标
- **error**: 红色左边框 + ✕ 图标
- **info**: 蓝色左边框 + ℹ 图标

#### 动画特性
- **进入动画**：从右侧滑入（300ms）
- **退出动画**：向右滑出（200ms）
- **进度条**：5秒倒计时动画
- **自动关闭**：无需用户操作
