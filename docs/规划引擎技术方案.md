## 一、设计目标

### 1.1 核心目标

实现一个**独立的规划引擎**,使云枢具备:
- ✅ 复杂任务的自动分解能力
- ✅ 多步骤任务的路径规划能力
- ✅ ReAct(Reasoning + Acting)循环执行能力
- ✅ 任务执行过程中的自适应调整能力
- ✅ 执行失败后的回溯和重试能力

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| **模块独立性** | 规划引擎独立于LLM,可单独测试和演进 |
| **接口抽象** | 遵循依赖倒置原则,依赖抽象而非实现 |
| **状态可追溯** | 每个规划步骤都有完整的执行记录 |
| **容错恢复** | 支持执行失败时的自动重试和回退 |
| **渐进增强** | 支持逐步增加规划能力的复杂度 |

### 4.1 任务分解器(TaskDecomposer)

#### 4.1.1 设计目标

将复杂的自然语言任务分解为可执行的任务图。

#### 4.1.2 分解策略

| 策略 | 适用场景 | 实现方式 |
|------|----------|----------|
| **LLM驱动分解** | 复杂开放式任务 | 调用LLM进行任务拆解 |
| **规则驱动分解** | 结构化任务 | 预定义任务模板匹配 |
| **混合分解** | 通用场景 | LLM + 规则组合 |

#### 4.1.3 分解流程

```

    ▼

└────────┬────────┘
         │
         ▼
┌─────────────────┐
```

**任务图生成算法**:

```python
class TaskGraphGenerator:
    """任务图生成器"""

    def generate(self, task_description: str, context: Dict) -> TaskGraph:
        """
        生成任务依赖图

        算法步骤:
        1. 使用LLM提取关键动作动词
        2. 识别动作宾语和条件
        3. 分析动作间的时间/逻辑依赖
        4. 构建有向无环图(DAG)
        5. 拓扑排序确定执行顺序
        """
        # Step 1: 动作提取
        actions = self._extract_actions(task_description)

        # Step 2: 依赖分析
        dependencies = self._analyze_dependencies(actions)

        # Step 3: 图构建
        graph = self._build_graph(actions, dependencies)

        # Step 4: 验证无环
        if not self._is_dag(graph):
            raise PlanningError("任务存在循环依赖,无法规划")

        # Step 5: 拓扑排序
        execution_order = self._topological_sort(graph)

        return TaskGraph(actions, dependencies, execution_order)

    def _extract_actions(self, description: str) -> List[Task]:
        """提取任务动作"""
        prompt = f"""
        分析以下任务描述,提取关键动作步骤:

        任务: {description}

        要求:
        1. 列出所有需要执行的动作
        2. 每个动作说明执行内容和目的
        3. 识别动作间的逻辑关系(顺序/并行/条件)
        4. 标注每个动作的前置条件和后置条件

        输出JSON格式:
        {{
            "actions": [
                {{
                    "id": "step_1",
                    "description": "动作描述",
                    "type": "atomic|sequential|parallel|conditional",
                    "preconditions": ["前置条件"],
                    "postconditions": ["后置条件"],
                    "related_tools": ["可能的工具"]
                }}
            ]
        }}
        """
        # 调用LLM解析
        response = self._llm.chat([{"role": "user", "content": prompt}])
        return self._parse_llm_response(response)
```

```python
class PlanExecutor:
    """计划执行引擎"""

    def __init__(self, tool_registry, max_retries=3):
        self.tool_registry = tool_registry
        self.max_retries = max_retries
        self.execution_history: List[ExecutionRecord] = []

    async def execute_plan(self, plan: Plan) -> ExecutionResult:
        """执行完整计划"""
        plan.state = PlanState.EXECUTING

        while not self._is_plan_complete(plan):
            # 获取下一个可执行任务
            next_task = self._get_next_executable_task(plan)
            if next_task is None:
                break

            # 执行任务
            result = await self._execute_task_with_retry(next_task)

            # 记录执行
            self._record_execution(plan, next_task, result)

            # 检查是否需要更新依赖
            self._update_dependencies(plan, next_task)

            # 检查是否需要反思
            if self._should_reflect(plan, next_task):
                await self._execute_reflection(plan)

        return self._generate_execution_result(plan)

    async def _execute_task_with_retry(self, task: Task) -> ExecutionResult:
        """带重试的任务执行"""
        for attempt in range(self.max_retries):
            try:
                # 检查前置条件
                if not self._check_preconditions(task):
                    return ExecutionResult(success=False,
                        error="前置条件不满足")

                # 执行动作
                action = self._determine_action(task)
                result = await self._execute_action(action)

                # 验证结果
                if self._validate_result(task, result):
                    return result
                else:
                    raise PlanningError(f"结果验证失败")

            except Exception as e:
                if attempt < self.max_retries - 1:
                    # 执行回滚
                    await self._rollback(task)
                    # 等待后重试
                    await asyncio.sleep(2 ** attempt)
                else:
                    return ExecutionResult(success=False, error=str(e))

        return ExecutionResult(success=False, error="重试次数耗尽")

    def _determine_action(self, task: Task) -> Action:
        """根据任务确定执行动作"""
        # 从工具注册表查找匹配的工具
        tool = self.tool_registry.find_tool(task.description)
        if tool:
            return Action(
                id=f"action_{task.id}",
                tool_name=tool.name,
                tool_params=self._extract_params(task),
                preconditions=task.constraints,
                postconditions=[],
            )
        else:
            # 降级为LLM执行
            return self._create_llm_fallback_action(task)
```

### 4.3 反思引擎(Reflector)

#### 4.3.1 设计目标

在任务执行过程中和完成后,进行效果评估和经验学习。

#### 4.3.2 反思类型

| 类型 | 触发时机 | 目的 |
|------|----------|------|
| **步骤反思** | 每个子任务完成后 | 评估当前步骤效果,决定下一步 |
| **计划反思** | 计划执行完成后 | 总结整体执行效果 |
| **错误反思** | 任务失败时 | 分析失败原因,提出改进建议 |
| **学习反思** | 任务完成后 | 将经验转化为可复用的知识 |

#### 4.3.3 实现代码

```python
class Reflector:
    """反思引擎"""

    def __init__(self, llm_service, memory_manager):
        self.llm = llm_service
        self.memory = memory_manager

    async def step_reflect(self, plan: Plan, task: Task,
                          result: ExecutionResult) -> ReflectionResult:
        """步骤级反思"""
        prompt = f"""
        作为云枢的反思引擎,分析当前执行步骤的效果。

        原始任务: {task.description}
        执行动作: {result.action}
        执行结果: {result.output}
        执行成功: {result.success}
        观察结果: {result.observation}

        请分析:
        1. 当前步骤是否达到预期目标?
        2. 如果继续执行,需要注意什么?
        3. 是否需要调整后续计划?

        输出JSON:
        {{
            "assessment": "评估结论",
            "next_action": "下一步建议",
            "adjustments": ["计划调整建议"],
            "confidence": 0.0-1.0
        }}
        """
        response = await self.llm.chat([{"role": "user", "content": prompt}])

        # 解析并执行建议
        suggestion = self._parse_suggestion(response)

        if suggestion.adjustments:
            await self._apply_adjustments(plan, suggestion.adjustments)

        return suggestion

    async def plan_reflect(self, plan: Plan) -> PlanReflection:
        """计划级反思 - 总结整个计划的执行"""
        execution_summary = self._summarize_execution(plan)

        prompt = f"""
        反思这次计划执行的完整过程:

        原始任务: {plan.original_task}
        执行摘要: {execution_summary}

        分析维度:
        1. 计划有效性: 原计划是否合理?
        2. 执行效率: 各步骤耗时是否合理?
        3. 决策质量: 每步决策是否正确?
        4. 经验总结: 有哪些可以改进的地方?

        输出JSON:
        {{
            "overall_score": 0.0-10.0,
            "effectiveness": "计划有效性评估",
            "efficiency": "执行效率评估",
            "lessons": ["经验教训"],
            "improvements": ["改进建议"],
            "knowledge_to_learn": ["需要学习的知识"]
        }}
        """
        response = await self.llm.chat([{"role": "user", "content": prompt}])

        reflection = self._parse_plan_reflection(response)

        # 保存到记忆
        await self._store_learning(plan.original_task, reflection)

        return reflection

    async def learn_from_experience(self, task: str,
                                   result: ExecutionResult) -> None:
        """从经验中学习"""
        if result.success:
            # 成功经验: 保存成功模式
            pattern = {
                "task_type": self._classify_task(task),
                "successful_actions": result.actions,
                "context": result.context,
                "success_rate": 1.0
            }
            await self.memory.save_pattern(pattern)
        else:
            # 失败经验: 保存失败教训
            lesson = {
                "task_type": self._classify_task(task),
                "failure_point": result.error,
                "attempted_actions": result.actions,
                "context": result.context
            }
            await self.memory.save_lesson(lesson)
```

#### 4.4.2 状态转换规则

```python
class PlanStateMachine:
    """计划状态机"""

    VALID_TRANSITIONS = {
        PlanState.INIT: {PlanState.DECOMPOSING, PlanState.CANCELLED},
        PlanState.DECOMPOSING: {PlanState.READY, PlanState.FAILED, PlanState.CANCELLED},
        PlanState.READY: {PlanState.EXECUTING, PlanState.CANCELLED},
        PlanState.EXECUTING: {PlanState.PAUSED, PlanState.COMPLETED,
                              PlanState.FAILED, PlanState.CANCELLED},
        PlanState.PAUSED: {PlanState.EXECUTING, PlanState.CANCELLED},
        PlanState.COMPLETED: set(),  # 终态
        PlanState.FAILED: set(),     # 终态
        PlanState.CANCELLED: set(),  # 终态
    }

    def transition(self, plan: Plan, new_state: PlanState,
                   reason: str = None) -> bool:
        """执行状态转换"""
        if new_state not in self.VALID_TRANSITIONS.get(plan.state, set()):
            raise InvalidStateTransitionError(
                f"无法从 {plan.state.value} 转换到 {new_state.value}"
            )

        old_state = plan.state
        plan.state = new_state
        plan.updated_at = datetime.now()

        # 记录转换
        self._record_transition(plan, old_state, new_state, reason)

        # 触发钩子
        self._trigger_hooks(plan, old_state, new_state)

        return True

    def _trigger_hooks(self, plan: Plan, old: PlanState, new: PlanState):
        """触发状态转换钩子"""
        hooks = {
            (PlanState.INIT, PlanState.DECOMPOSING): self._on_decompose_start,
            (PlanState.DECOMPOSING, PlanState.READY): self._on_plan_ready,
            (PlanState.EXECUTING, PlanState.COMPLETED): self._on_execution_complete,
            (PlanState.EXECUTING, PlanState.FAILED): self._on_execution_failed,
            (PlanState.EXECUTING, PlanState.PAUSED): self._on_execution_paused,
            (PlanState.EXECUTING, PlanState.CANCELLED): self._on_execution_cancelled,
        }

        hook = hooks.get((old, new))
        if hook:
            asyncio.create_task(hook(plan))
```

## 五、ReAct循环实现

### 5.1 ReAct算法

```python
class ReActLoop:
    """
    ReAct (Reasoning + Acting) 循环实现

    核心思想: 在推理和行动之间交替,逐步完成任务
    """

    def __init__(self, planner: PlanningCore, reflector: Reflector,
                 max_iterations=10):
        self.planner = planner
        self.reflector = reflector
        self.max_iterations = max_iterations

    async def run(self, task: str, context: Dict = None) -> ReActResult:
        """
        执行ReAct循环

        伪代码:
        while steps < max_iterations:
            1. Thought: 思考当前状态,决定下一步行动
            2. Action: 执行选定的行动
            3. Observation: 观察行动结果
            4. 如果任务完成,退出循环

        Returns:
            ReActResult: 包含执行结果和推理轨迹
        """
        context = context or {}
        steps = []
        history = []

        for iteration in range(self.max_iterations):
            # ── 1. Thought (思考) ──
            thought = await self._think(task, context, history)

            if thought.action_type == "finish":
                # 任务完成
                return ReActResult(
                    success=True,
                    result=thought.result,
                    steps=steps,
                    iterations=iteration + 1
                )

            # ── 2. Action (行动) ──
            action_result = await self._act(thought.action, context)

            # ── 3. Observation (观察) ──
            observation = self._observe(action_result)

            # 记录步骤
            step = ReActStep(
                iteration=iteration,
                thought=thought.reasoning,
                action=thought.action,
                observation=observation,
                success=action_result.success
            )
            steps.append(step)
            history.append(step)

            # ── 4. 反思检查 ──
            if self._should_reflect(action_result):
                reflection = await self.reflector.step_reflect(
                    task, thought.action, action_result
                )
                if reflection.adjustments:
                    # 应用调整
                    context.update(reflection.adjustments)

            # 检查是否陷入循环
            if self._detect_loop(steps):
                raise PlanningError("检测到执行循环,无法继续")

        # 达到最大迭代次数
        return ReActResult(
            success=False,
            result="达到最大迭代次数,任务未完成",
            steps=steps,
            iterations=self.max_iterations
        )

    async def _think(self, task: str, context: Dict,
                    history: List[ReActStep]) -> ThoughtResult:
        """思考: 分析当前状态,决定下一步行动"""
        history_text = self._format_history(history)
        context_text = self._format_context(context)

        prompt = f"""
        作为云枢的思考引擎,分析当前状态并决定下一步行动。

        当前任务: {task}

        执行历史:
        {history_text}

        当前上下文:
        {context_text}

        可用工具:
        - check_health: 检查身体状态
        - search_memory: 搜索记忆
        - get_sensor_summary: 获取传感器摘要
        - web_search: 网络搜索
        - llm_chat: 与用户对话

        思考过程:
        1. 分析当前状态:我们已经完成了什么?
        2. 识别目标: 距离完成任务还差什么?
        3. 选择行动: 下一步应该做什么?
        4. 制定计划: 具体如何执行?

        输出JSON:
        {{
            "reasoning": "详细推理过程",
            "action_type": "tool_call|response|finish|ask_user",
            "action": {{
                "tool": "工具名",
                "params": {{"参数"}},
                "description": "行动描述"
            }},
            "confidence": 0.0-1.0,
            "result": "如果action_type是finish或response,这里放结果"
        }}
        """
        response = await self.planner.llm.chat([
            {"role": "user", "content": prompt}
        ])
        return self._parse_thought(response)

    async def _act(self, action: Action,
                   context: Dict) -> ExecutionResult:
        """执行行动"""
        if action.tool_name == "llm_chat":
            # 直接回复用户
            return ExecutionResult(
                success=True,
                output=action.tool_params.get("response"),
                observation=f"已回复用户"
            )
        else:
            # 调用工具
            tool = self.planner.tool_registry.get(action.tool_name)
            if not tool:
                return ExecutionResult(
                    success=False,
                    error=f"未找到工具: {action.tool_name}"
                )

            return await tool.execute(**action.tool_params)

    def _format_history(self, history: List[ReActStep]) -> str:
        """格式化执行历史"""
        if not history:
            return "(无历史,这是第一步)"

        lines = []
        for step in history[-5:]:  # 只显示最近5步
            lines.append(f"- 步骤{step.iteration}: {step.thought}")
            lines.append(f"  行动: {step.action}")
            lines.append(f"  结果: {step.observation}")
        return "\n".join(lines)
```

```python
class DigitalLife:

        """
        扩展输入处理流程

        新流程:
        1. 感知身体状态
        2. 判断是否需要规划
        3a. 简单任务 → 直接执行
        3b. 复杂任务 → 调用ReAct循环
        4. 反思总结
        """
        readings = self.check_health()
        body_status = self._build_body_status(readings)

        # 判断任务复杂度
        complexity = self._assess_complexity(user_input)

        if complexity == "simple":
            # 简单任务直接执行
            response = self._call_llm_simple(user_input, body_status)
        else:
            # 复杂任务启用规划
            result = self._planner.chat(user_input, context={
                "body_status": body_status,
                "behavior_mode": self._current_mode.value
            })
            response = result.response

        # 反思
        if self._behavior.profile.enable_reflection:
            self.self_reflect(user_input, response)

        # 记忆
        self._memory.add_message("user", user_input)
        self._memory.add_message("assistant", response)

        return response

    def _assess_complexity(self, task: str) -> str:
        """评估任务复杂度"""
        # 简单启发式判断
        simple_indicators = ["?", "吗", "什么", "多少", "帮我"]
        complex_indicators = ["帮我完成", "帮我创建", "帮我分析整个",
                            "帮我构建", "流程", "系统"]

        complex_score = sum(1 for i in complex_indicators if i in task)
        simple_score = sum(1 for i in simple_indicators if i in task)

        if complex_score > simple_score:
            return "complex"
        return "simple"

    async def execute_planned_task(self, task: str) -> str:
        """执行有规划的任务"""
        result = await self._react_loop.run(task)
        return result.result
```
