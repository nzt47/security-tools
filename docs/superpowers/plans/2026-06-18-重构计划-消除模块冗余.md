# 云枢模块冗余重构计划

## 概述
消除项目中因多次迭代遗留的新老版本并存问题，涉及 6 个步骤、约 40-50 个文件。

| 类别 | 冗余行数 | 涉及文件数 |

| Persona 新老并存 | ~1,200 | 5 → 2 |
| 缓存碎片化 | ~800 | 3 → 2 |
| 日志 emoji 重复 | ~60 | 2 → 1 |
| 监控碎片 | ~1,000 | 8 → 4 |

| 测试文件碎片 | ~12,000 | 40+ → 15 |
| **总计** | **~16,000** | **~60 → ~25** |

## 步骤一：Persona 模块合并

**涉及文件**：persona/persona_model.py → 删除，persona/distillation.py → 删除，persona/distiller.py → 删除

**操作**：
1. 读取 `persona/persona_model.py`（256行）和 `persona/persona_model_enhanced.py`（518行）
   - 两个文件定义了完全相同的 `PersonaLayer` 类
   - `PersonaModel` vs `PersonaModelEnhanced`：增强版增加了 to_dict/merge/diff 方法
   - **操作**：保留增强版，删除基础版，添加 `PersonaModel = PersonaModelEnhanced` 别名

2. 读取 `persona/distillation.py`（372行）和 `persona/distillation_enhanced.py`（567行）
   - 两个文件定义了完全相同的 `PersonalityPreferenceExtractor` 类
   - 增强版增加了情感分析、置信度、衰减机制
   - 读取 `persona/distiller.py`（502行）— 第三套蒸馏逻辑
   - **操作**：将 distiller 核心逻辑并入增强版，删除基础版和 distiller

3. 更新 `persona/__init__.py` 的导出

4. 更新所有 `from persona.persona_model import` → `from persona.persona_model_enhanced import`
   更新所有 `from persona.distillation import` → `from persona.distillation_enhanced import`

**验证**：
```bash
python -m pytest tests/unit/test_persona_model.py tests/unit/test_distiller.py tests/unit/test_persona_injector.py tests/unit/test_persona_boundary_cases.py agent/tests/test_persona.py -x -q --tb=short
python -c "from persona import PersonaModel, PersonaLayer, PersonalityPreferenceExtractor; print('OK')"
```
