## ADR 管理规范

### 新增 ADR 流程
1. 复制 `adr-template.md` 创建新文档
2. 文件命名规则: `NNN-{descriptive-title}.md`
3. 填写所有必填字段
4. 更新本目录索引
5. 提交评审

### 状态定义
- **待评审** - 已提交，等待评审
- **已接受** - 已采纳并将实施
- **已拒绝** - 已被拒绝，不再考虑
- **已废弃** - 已废弃的决策
- **已替代** - 被新的ADR替代

### 编号规则
- 3位数字，从001开始递增
- 编号永不复用
