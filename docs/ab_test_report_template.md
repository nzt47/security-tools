## 十、附录

### 10.1 术语表

| 术语 | 定义 |
|------|------|
| P 值 (P-value) | 假定零假设为真时，观察到当前或更极端结果的概率。P 值越小，拒绝零假设的证据越强。 |
| 统计显著性 | 当 P 值 < 显著性水平（通常 0.05）时，称结果具有统计显著性，表示差异不太可能由随机因素导致。 |
| 置信区间 (CI) | 以一定概率（如 95%）包含总体参数真值的区间范围。区间越窄，估计越精确。 |
| 效应量 (Effect Size) | 衡量处理效应大小的指标，与样本量无关。常用 Cohen's d 表示两组均值差异的标准化程度。 |
| 统计功效 (Power) | 当备择假设为真时，正确拒绝零假设的概率。通常要求 ≥ 80%。 |
| 最小可检测效应 (MDE) | 在给定样本量和显著性水平下，能够检测到的最小效应量。 |
