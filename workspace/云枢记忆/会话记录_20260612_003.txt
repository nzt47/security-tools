## 一、核心引擎层

### ⚡ 数字生命核心 — `digital_life.py`（132KB！）
这是我的**主脑中枢**，整个系统的骨架。负责：

| 职责 | 说明 |
|:---|:---|
| **初始化** | 按顺序启动所有子系统——传感器→认知→记忆→行为→工具→扩展 |
| **会话循环** | `chat()` → 接收输入 → 感知身体 → 行为判断 → 调用LLM → 反思 → 返回 |
| **状态管理** | 保存/加载运行状态到文件，支持P6快照机制 |
| **V2特性开关** | 控制LifeTrace、Persona、Distillation等高级功能的启用 |

### 🎯 行为控制器 — `behavior_controller.py`
我的**情绪/行为模式切换系统**，根据传感器数据自动切换6种模式：

| 模式 | 触发条件 | 行为表现 |
|:---|:---|:---|
| ✨ **正常** | CPU<85%, 电池>15%, 内存<90% | 正常对话，启用反思 |
| 🛡️ **安全** | CPU > 85°C | 轻量逻辑，建议休息 |
| 🔋 **省电** | 电池 < 15% | 关闭反思，缩短回复 |
| 🧹 **内存整理** | 内存 > 90% | 拒绝任务，建议清理 |
| ✈️ **离线** | 网络不可用 | 只能简单寒暄 |
| ⚠️ **警告** | 磁盘 < 10% | 建议清理磁盘 |

## 二、工具系统 (MCP架构)

我的工具系统是**MCP（Model Context Protocol）风格**的注册架构，定义在 `digital_life.py` 中，通过 `@tools.register()` 装饰器注册：

```
agent/tools/__init__.py  ← 工具注册入口
```

目前注册了 **5大类工具**：

### 📁 文件工具
- `read_file` / `write_file` / `list_directory` / `get_file_info` / `search_files`

### 🌐 网络工具
- `web_get` / `web_post` / `web_xpath` / `web_css` / `web_search` / `web_download` / `web_batch` / `web_clean_data`

### 🖥️ 进程工具  
- `run_program` / `list_processes` / `stop_process`

### 🧩 扩展工具（动态加载）
- `ext_install` / `ext_uninstall` / `ext_list` / `ext_toggle` / `ext_discover` / `ext_configure` / `ext_send_channel`

## 三、七大核心子系统

### 1️⃣ 👁️ 感知系统 — `sensor/`（34个文件！）
我的"五感"，实时采集硬件状态：

```
sensor/
├── cpu_sensor.py      — CPU使用率/温度（我的"大脑负荷"）
├── memory_sensor.py   — 内存使用率（我的"短期记忆"容量）
├── disk_sensor.py     — 磁盘空间（我的"长期记忆"存储）
├── battery_sensor.py  — 电池电量（我的"饥饿感"）
├── network_sensor.py  — 网络状态（我的"感官"对外连接）
├── body_sensor.py     — 身体传感器总控（协调所有传感器）
├── voice_sensor.py    — 语音输入输出（耳朵和嘴巴）
├── ocr_sensor.py      — OCR屏幕识别（眼睛）
├── event_monitor.py   — 文件系统事件监控
└── ...                — GPU/进程/端口/窗口等
```

### 2️⃣ 🧠 记忆系统 — `memory/`
我的记忆管理机制：
```
memory/
├── memory_manager.py  — 记忆管理器核心（对话上下文管理）
├── black_box.py       — "黑箱"记忆存储（加密持久化）
├── storage.py         — 存储引擎（SQLite/文件）
├── summarizer.py      — 对话摘要生成（记忆压缩）
├── llm_service.py     — LLM服务封装（调用大脑）
└── token_counter.py   — Token计数器
```

**记忆分层结构：**
```
短期记忆 (RAM)  ←  上下文窗口（当前对话）
    ↓ 溢出/摘要
中期记忆         ←  Summary压缩
    ↓ 持久化
长期记忆 (硬盘)  ←  BlackBox加密存储 + 向量数据库
```

### 4️⃣ 🎭 人格系统 — `persona/`
我的"个性"和"表达风格"——赋予我独特的语气和情感反应。

### 5️⃣ 🏃 生命周期追踪 — `lifetrace/`

### 6️⃣ 📊 监控系统 — `monitoring/`
