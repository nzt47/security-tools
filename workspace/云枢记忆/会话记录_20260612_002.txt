## 📂 技能存放位置一览

### 1️⃣ 技能配置文件（JSON 清单）
| 文件路径 | 说明 |
|---------|------|
| `agent/data/skills.json` | 📋 主技能配置文件 |
| `data/skills.json` | 📋 辅助技能配置 |

这两个文件里记录了每个技能的 **ID、名称、启用状态、描述和参数**。比如：

```json

  "id": "self_reflection",
  "name": "自省反思",
  "enabled": true,
  "params": { "frequency": "always", "depth": "normal" }
}
```
