# 技能安装安全检查系统 - 更新日志

## 更新时间
2026-06-23

## 更新概述
本次更新为技能安装流程添加了全面的安全检查机制，包括代码威胁扫描、权限验证、兼容性检测和完整性验证。

---

## 新增功能

### 1. 安全检查器 (`agent/extensions/security_checker.py`)

**安全评估功能：**
- 代码威胁扫描：检测危险代码模式（subprocess执行、文件操作、网络访问等）
- 权限验证：检查技能描述中的敏感权限关键词
- 数据合规性检查：检测数据收集相关操作
- 安全等级分级：PASS（安全）、WARNING（警告）、BLOCK（阻止）
- 安全分数评分（0-100分）

**兼容性分析：**
- 检测与已安装技能的名称冲突
- 检测功能重叠（文件操作、网络、记忆、语音、安全、搜索等）
- 检测与系统原生功能的冲突

### 2. 安全检查技能 (`agent/extensions/security_check_skill.py`)

- 公开接口供用户手动触发安全检查
- 格式化的安全检查报告生成

### 3. 安装引擎增强 (`agent/extensions/installer.py`)

**ZIP 完整性验证：**
- 使用 `testzip()` 验证 ZIP 文件完整性
- 检测损坏或无效的 ZIP 文件
- 检查是否包含必要的 Python 文件

**GitHub 下载流程优化：**
- 详细日志输出，便于排查网络或下载失败
- 支持三种下载方式（gh CLI、git clone、ZIP 下载）
- 错误处理增强（HTTP错误、网络错误、超时等）

**压缩包自动解压：**
- 支持 `.zip` 和 `.tar.gz` 格式
- 自动跳过 GitHub ZIP 的第一层目录
- 递归解压所有子目录

### 4. 技能安装器集成 (`agent/extensions/skills_installer.py`)

- 在安装流程中自动触发安全检查
- 根据安全等级决定是否允许安装
- 详细的日志输出（WARNING/BLOCK 级别）
- 自动清理被拦截技能的临时文件

## 安全等级说明

| 安全等级 | 分数范围 | 能否安装 | 说明 |
|---------|---------|---------|------|
| PASS | 91-100 | ✓ 可以 | 未发现安全问题 |
| WARNING | 41-90 | ✓ 可以 | 存在潜在风险，建议审查 |
| BLOCK | 0-40 | ✗ 阻止 | 存在严重安全风险 |

---

## 检测的危险模式

### 高风险
- subprocess 执行（`subprocess.run`, `subprocess.call` 等）
- OS 命令执行（`os.system`, `os.popen`）
- 文件删除操作（`shutil.rmtree`, `os.remove`）

### 中风险
- 网络请求（`requests.get/post`, `urllib`）
- 网络套接字（`socket.socket`）
- 文件写入操作（`open(file, 'w')`）

### 低风险
- 环境变量访问（`os.environ`）
- 数据序列化（`pickle`, `marshal`）

---

## 兼容性检测

### 功能类别
- 文件操作（文件、文件系统、读写等）
- 网络功能（网络、network、HTTP等）
- 记忆存储（记忆、memory、存储等）
- 语音功能（语音、voice、音频等）
- 安全功能（安全、security、加密等）
- 搜索功能（搜索、search、查询等）
- 任务规划（规划、plan、调度等）
- 网页浏览（网页、browser、浏览器等）
- 监控功能（监控、monitor、日志等）

### 冲突类型
- **名称冲突**：技能名称与已安装技能相同 → 阻止安装
- **功能重叠**：技能功能与已安装技能重叠 → 警告但允许安装
- **系统功能重叠**：技能功能与系统原生功能重叠 → 警告

## 安装流程

```
用户请求安装技能
        ↓
解析来源（github:/url:/local:/npm:/pip:）
        ↓
获取技能包
        ↓
自动解压（如需要）
        ↓
安全检查
        ├─ 代码扫描
        ├─ 权限验证
        ├─ 兼容性检测
        └─ 生成安全报告
        ↓
安全等级判定
        ├─ PASS/WARNING → 继续安装
        └─ BLOCK → 拦截并清理临时文件
        ↓
注册到扩展存储
```
