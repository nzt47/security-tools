浮层面板规则

  • 点击图标 → 在图标右侧弹出 240px 浮层面板
  • 点击浮层外部 / ✕ 按钮 / 再次点击图标 → 关闭
  • 同一时间只有一个浮层打开
  • 浮层内容与当前侧边栏面板内容相同

☰ 完整侧边栏

  • 点击 ☰ → 展开 280px 完整侧边栏（替代图标栏）
  • 再次点击 ☰ → 收缩回 48px 图标栏
  • 状态通过 sessionStorage 保持
  • 展开后 5 个模块显示完整标签和面板

面板显示规则

  • 对话视图：右侧面板始终可见
  • 全景视图：右侧面板隐藏，全景内容全宽显示
  • 10 秒自动刷新面板数据
  • 与顶栏指标保持数据同步

响应式处理

  • 窗口 < 900px：右侧面板折叠到底部或隐藏
  • 窗口 < 768px：图标栏隐藏，☰ 移入顶栏
  • 窗口 < 600px：全景导航折叠为顶部 tab
  • 所有自适应通过 CSS media query 实现
