🧪 存储验证测试页面

测试说明

⚠️ 请先打开浏览器开发者工具 (F12) 查看控制台输出

1. 点击下方按钮切换存储模式

2. 观察控制台输出

3. 测试完成后可关闭此标签页验证 sessionStorage 清除

操作按钮

测试步骤

【sessionStorage 清除验证步骤】

1. 点击 "切换到 sessionStorage" 按钮
2. 点击 "运行所有测试" 按钮
3. 观察控制台输出，确认数据已保存
4. 按 F5 刷新页面 → 数据应该保留 ✓
5. 关闭此标签页
6. 重新打开此 URL → 数据应该清除 ✓

【localStorage 清除验证步骤】

1. 点击 "切换到 localStorage" 按钮
2. 点击 "运行所有测试" 按钮
3. 关闭此标签页
4. 重新打开此 URL → 数据应该保留 ✓

路由参数测试

【URL 参数格式】
?storage=localStorage
?storage=sessionStorage

【示例】
http://localhost:5173/?storage=localStorage
http://localhost:5173/?storage=sessionStorage
